# Confile
A properties file editor

| Command | Description                                     |       Example       |
|:-------:|-------------------------------------------------|:-------------------:|
|    /m   | show menu                                       |          -          |
|    /s   | show properties                                 |          -          |
|    /l   | load properties                                 |          -          |
|    /#   | comment property (undo if already is commented) | /# [property_index] |
|    /f   | find property                                   |  /f [search_string] |
|    /o   | open file                                       |      /o [path]      |
|    /x   | save file                                       |          -          |
|    /c   | clear screen                                    |          -          |
|    /q   | quit program                                    |          -          |
